# Erika Noonan - Portfolio

Professional portfolio website for Erika Noonan, Senior Technical Program Manager specializing in global change governance, cloud & security.

## Build & Deployment

This site is built with [Hugo](https://gohugo.io/) and hosted on GitHub Pages.

### Local Development

```bash
hugo server
```

Site will be available at `http://localhost:1313`

### Build for Production

```bash
hugo
```

### Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.
